# Proyecto

## Información del Curso
- **Curso:** Curso: React Avanzado

## Descripción del Proyecto
TaskFlow es una aplicación web para la gestión de tareas y proyectos. Permite a los usuarios crear, visualizar y organizar tareas, asignar prioridades y estados, y gestionar proyectos de manera eficiente. El objetivo es practicar conceptos avanzados de React, TypeScript y arquitectura de componentes moderna.

## Arquitectura del Proyecto
- **React + TypeScript:** Estructura basada en componentes funcionales y tipado estricto.
- **Vite:** Herramienta de build y desarrollo rápido.
- **Estructura de carpetas:**
  - `src/features/taskManagement/`: Lógica y componentes para gestión de tareas.
  - `src/features/projectsManagement/`: Lógica y componentes para gestión de proyectos.
  - `src/shared/`: Componentes reutilizables y UI.
  - `src/assets/`: Recursos estáticos.
- **Hooks personalizados:** Para lógica de negocio reutilizable.
- **Separación de UI y lógica:** Componentes atómicos, moleculares y de layout.

## Instrucciones de Instalación
1. Clona el repositorio:
   git clone [URL_DEL_REPOSITORIO]
   cd taskflow
2. Instala las dependencias:
   npm install
3. Inicia el servidor de desarrollo:
   npm run dev
